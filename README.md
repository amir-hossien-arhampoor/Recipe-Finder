# 🍴 Recipe Finder App

A simple and interactive web app to search and explore recipes based on ingredients.  
Built with **React, HTML, CSS, and JavaScript**.

---

## 🚀 Features
- Search recipes by entering ingredients.
- also searching recipes by category is available.
- Responsive design for mobile and desktop.
- Clean and user-friendly UI.

---

## 🛠️ Tech Stack
- React
- JavaScript (ES6+)
- HTML5, CSS3
- OPEN AI
